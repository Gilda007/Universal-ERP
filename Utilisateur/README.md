# Système de Gestion de Courrier MINDCAF

Application de gestion de courrier pour le Ministère des Domaines, du Cadastre et des Affaires Foncières.

## Fonctionnalités

- Gestion des courriers (RTVS, DVC)
- Interface administrateur et utilisateur
- Système de suivi des documents
- Gestion des utilisateurs
- Rapports et analyses

## Installation

1. Cloner le repository
\`\`\`bash
git clone https://github.com/votre-username/nom-du-repo.git
\`\`\`

2. Installer les dépendances
\`\`\`bash
pip install -r requirements.txt
\`\`\`

3. Configurer la base de données MySQL

4. Lancer l'application
\`\`\`bash
python main.py
\`\`\`

## Structure du projet

- \`Administrateur/\` - Module administrateur
- \`Utilisateur/\` - Module utilisateur
- \`connexion_db/\` - Gestion de la base de données
- \`Frontends/\` - Interfaces utilisateur

## Technologies utilisées

- Python
- PyQt5/PySide2
- MySQL
- Qt Designer
